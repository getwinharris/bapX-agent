# Codex macOS App Design — Reference Notes (May 24, 2026)

## Layout Architecture
- 3-column design: Left sidebar | Center chat | Right panel
- Left sidebar: 48px collapsed, 240px expanded (icon rail pattern)
- Right panel: 380px when open, 0 when collapsed (300px on mobile)
- Resizer handle between center and right panel

## Left Sidebar
- Top: Logo/ brand mark
- Middle: Navigation items with SVG icons (vertical)
  - Chat, Projects, Deploy, Integrations, Skills, Settings
- Bottom: User avatar + name
- Labels hidden when collapsed, shown when expanded
- Inter font, 13px/500 weight for nav labels

## Chat Area
- Header: Title + session selector + panel toggle
- Messages: scrollable, user right-aligned/accent, assistant left-aligned/dark
- Input: textarea + send button at bottom
- Typing indicator: 3 bouncing dots
- Empty state: centered icon + heading

## Right Panel
- Tabs: Flow, Browser, Terminal
- Flow: step list with status grouping
- Browser: URL bar + nav buttons + iframe
- Terminal: monospace output + input

## Color Tokens (dark theme)
- Background: #0a0a0f (main), #121218 (secondary), #1a1a24 (tertiary)
- Text: #e8e8f0 (primary), #a1a1aa (secondary), #52525b (muted)
- Accent: #8b5cf6 (purple), #7c3aed (hover)
- Green: #22c55e, Red: #ef4444
- Border: rgba(255,255,255,0.06)
- Font: 'Inter', system-ui, sans-serif
