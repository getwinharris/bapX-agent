# Data Sources

> Access premium third-party data APIs integrated into Manus

Data Sources provide access to premium third-party data APIs that are integrated into Manus. These data sources allow you to enrich your tasks with real-time information from external platforms without managing API keys or integrations yourself.

<img src="https://mintcdn.com/docs-manus/ntt3_UJfi_wfMGAs/images/data-sources_indicator.png?fit=max&auto=format&n=ntt3_UJfi_wfMGAs&q=85&s=1b372e43a9a295a462e7d2c947509628" alt="Data Source in sessions" width="1238" height="647" data-path="images/data-sources_indicator.png" />

Simply mention the data you need in your task prompts, and Manus will automatically use the appropriate data source.

## How to Use Data Sources

Data sources are automatically available in your tasks. Simply mention the data you need in your prompt, and Manus will use the appropriate provider.

## Notes

<Info>
  All data sources are included with your Manus subscription. No additional API keys or setup required.
</Info>

<Note>
  Data availability and rate limits vary by provider and may be subject to additional restrictions. Manus automatically handles API calls and caching for optimal performance.
</Note>

<Warning>
  Data from third-party sources is subject to their terms of service and data freshness policies. Always verify critical information from official sources.
</Warning>
