# Editing and Previewing

> Manus lets you build and refine your web app with instant feedback and natural-language edits

### Edit: Effortless, Conversational Changes

Editing your web application in Manus is as simple as having a conversation. There is no need to write code or navigate complex menus. You can make changes to your site by describing what you want in natural language, from single tweaks to large-scale updates.

#### Visual Editing

Manus 1.5 introduced 3 flexible ways to edit your websites:

### **Quick Style Controls**

Click any element to open an instant style panel. No waiting, no prompts—just direct control:

* Colors: Change backgrounds, text, borders
* Typography: Adjust fonts, sizes, weights
* Spacing: Modify padding, margins, gaps
* Borders: Update styles, widths, radius

Perfect for rapid design iteration and fine-tuning.

### **Prompt-Based Batch Editing**

To make your workflow even more efficient, Manus allows you to perform **prompt-based group edits**. You can describe multiple changes in a single prompt, and Manus will intelligently package and apply them together as a single update. This is perfect for making a series of related adjustments without having to issue commands one by one.

For example, you can give a single, comprehensive instruction like this:

> "Okay, let's make a few changes. First, make the main heading bolder and change its color to our brand's dark blue. Next, adjust the spacing in the pricing table to be more generous. Finally, change the background color of the entire footer section to a light gray."

Manus will understand the distinct requests within your prompt and execute all of them in one go, saving you time and effort.

| Edit Type     | Example Prompt                                                  |
| :------------ | :-------------------------------------------------------------- |
| **Styling**   | `"Change the font of all headings to Georgia."`                 |
| **Layout**    | `"Increase the spacing between the items in this list."`        |
| **Content**   | `"Change the text of the main button to 'Get Started Now'."`    |
| **Structure** | `"Duplicate this section and place it below the contact form."` |

### AI Edit: Your Creative Partner

For more complex or creative changes, you can use the AI edit feature. Simply select an element and ask Manus to improve it. This is like having a design consultant and copywriter at your fingertips.

| Goal               | Example Prompt                                                                      |
| :----------------- | :---------------------------------------------------------------------------------- |
| **Improve Copy**   | `"Rewrite this headline to be more engaging and benefit-oriented."`                 |
| **Enhance Design** | `"Suggest a better, more modern color palette for this section."`                   |
| **Refine Layout**  | `"Restructure this feature list to be more visually appealing and easier to scan."` |

Manus will provide suggestions and apply the changes for you, making it easy to elevate your design and content beyond the quality of your content.

Manus provides a seamless experience for editing and previewing your web application in real-time. This allows for rapid iteration and ensures that you can visualize changes as you make them.

## Preview: See Your Application in Real-Time

Manus provides a live, interactive preview of your web application throughout the entire development process. This allows you to see your changes as they happen and test your application's functionality in a real-world environment.

### Desktop/Mobile

The preview window allows you to toggle between desktop and mobile views, ensuring your application is responsive and looks great on all devices. You can test the user experience on different screen sizes and make adjustments as needed to optimize for both desktop and mobile users.

### Path

The preview window includes a URL bar that displays the current path of the page you are viewing. This is useful for navigating to specific pages within your application and for understanding the URL structure of your site.

### Fullscreen

For a more immersive experience, you can expand the preview to fullscreen mode. This is particularly useful for presenting your application to stakeholders or for getting a better sense of the overall look and feel of your site without the distraction of the surrounding interface.

#### Live Preview

A live, interactive preview of your application is available from the very beginning of the development process. This allows you to see the results of your requests instantly and provide feedback for adjustments. The live preview is an essential part of the iterative development process, enabling you to refine your application with natural language commands and see the changes reflected immediately.
