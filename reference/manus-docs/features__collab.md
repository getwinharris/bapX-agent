# Manus Collab

Manus Collab is a real-time collaboration feature that enables teams to work together on Manus tasks. Instead of juggling multiple document versions, scattered email feedback, and endless "latest\_final\_v3" files, Collab provides one shared workspace where your entire team can prompt, edit, and refine work simultaneously.

When you enable collaboration on a task, you create a single source of truth accessible via one link. Everyone sees updates instantly, everyone can interact with Manus directly, and only the task owner consumes credits.

## Key Features

### Only Owner Consumes Credits

**Important**: In collaboration mode, only the task owner's credits are consumed. Collaborators can participate fully without using their own credits.

This makes Collab ideal for:

* Teams with mixed subscription levels
* Collaborating with clients or external partners
* Onboarding new team members
* Cross-functional projects

### Owner-Controlled Invitations

Only the task owner can invite collaborators. Collaborators cannot invite additional people.

This maintains control over:

* Who has access to the task
* Sensitive information visibility
* Credit consumption responsibility

## Important Notes

### Privacy and Content Sharing

* **Do not share personal information** without permission during collaboration
* **Do not share third-party content** without proper authorization
* **Comply with Terms of Use**: All participants must follow Manus platform policies

### Sandbox Login Cookies

**Important**: When the first collaborator joins a task, the sandbox environment login cookies are cleared.

**What this means**:

* If the task involves authenticated websites (e.g., logged into a CRM), you'll need to log in again
* This is a security measure to protect your credentials
* Only affects tasks using Browser Operator or Cloud Browser with logins

**Workaround**: Complete authenticated portions of the task before inviting collaborators, or be prepared to re-authenticate.

## When to Use Collab

**Ideal For**:

* Team content creation (presentations, reports, articles)
* Client co-creation and feedback
* Cross-functional projects requiring multiple perspectives
* Iterative design and refinement
* Training and onboarding (show how Manus works)
* Research projects with multiple contributors

**Not Ideal For**:

* Solo tasks that don't require input
* Highly sensitive work requiring strict access control
* Tasks where version history tracking is critical
* Simple, quick requests

## Tips for Effective Collaboration

**Set clear roles**:

* Designate who handles which aspects (content, design, data)
* Owner should guide overall direction
* Collaborators focus on specific contributions

**Use specific prompts**:

* ✅ "Add a slide about Q4 sales performance with the numbers from the spreadsheet"
* ❌ "Make it better"

**Communicate in the task**:

* Use prompts to explain changes: "Adding customer testimonials because..."
* This creates a clear record of decisions

**Invite strategically**:

* Start with core team members
* Add others as needed
* Remember: owner controls invitations

**Be mindful of timing**:

* Coordinate major changes to avoid conflicts
* Use async collaboration for different time zones
* Real-time sessions for rapid iteration

## Common Questions

<AccordionGroup>
  <Accordion title="Can collaborators invite others? ">
    No. Only the task owner can invite collaborators. This maintains control over access and credit consumption.
  </Accordion>

  <Accordion title="Do collaborators need a Manus account? ">
    Yes. Collaborators receive an invitation and need to sign in to Manus to access the task.
  </Accordion>

  <Accordion title="Can I remove a collaborator? ">
    Yes. The task owner can manage collaborator access through the sharing settings.
  </Accordion>

  <Accordion title="Can collaborators see the entire task history? ">
    Yes. All collaborators see the complete task history, including prompts and outputs from before they joined.
  </Accordion>

  <Accordion title="What if two people prompt Manus at the same time? ">
    Manus processes prompts sequentially. Both will be completed, but one after the other.
  </Accordion>

  <Accordion title="Can collaborators export the results?">
    Yes. All collaborators can access and export the task outputs.
  </Accordion>

  <Accordion title="Does Collab work with all Manus features? ">
    Yes. Collab works with slides, web development, research, data analysis, and all other Manus capabilities.
  </Accordion>
