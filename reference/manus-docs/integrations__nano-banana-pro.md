# Nano Banana Pro

> AI-powered image generation for visually stunning slides and graphics

## What is Nano Banana Pro?

Nano Banana Pro is Google's advanced image generation model, integrated directly into Manus for creating professional-quality visuals. When you need slides or images that look like they were designed by a studio, Nano Banana Pro generates custom, high-resolution imagery tailored specifically to your content.

This transforms Manus from a text-based tool into a complete visual content creation platform, capable of producing presentation decks and graphics that would typically require professional design software and expertise.

## Why Use Nano Banana Pro?

Most AI image generators struggle with two critical problems: they can't render readable text within images, and they can't maintain visual consistency across multiple outputs. This makes them impractical for professional presentations.

Nano Banana Pro solves both:

* Nano Banana's text rendering: Crisp, legible text integrated directly into images
* Manus's contextual understanding: Consistent artistic direction across entire decks

This enables professional-quality visual content without design skills or expensive software

### How Manus Generates Your Content

For slide decks, Manus:

1. Analyzes your prompt and understands your message
2. Creates a complete presentation outline
3. Determines visual direction (colors, layout, imagery)
4. Batch-generates all slides with consistent design
5. Delivers a ready-to-use deck

For standalone images, Manus generates custom visuals in moments.

### 4. Export and Use

Download your slides as PDF or individual images, ready for any platform.

### Use Dynamic Prompts

Combine Nano Banana Pro with other Manus features for powerful, data-driven presentations:

**Wide Research + Slides:**

```
Analyze 50 SaaS pricing pages, identify patterns, then create a pricing strategy presentation in a professional B2B style with data visualizations.
```

**Data Analysis + Slides:**

```
Analyze this sales data CSV, identify trends and insights, then create a quarterly business review deck using corporate colors and clean charts.
```

**Manus Browser Operator + Slides:**

```
Visit these 5 competitor websites, extract their key features and pricing, then create a competitive analysis deck using a modern, minimalist design.
```

**Scheduled Tasks + Slides:**

```
Every Monday at 9 AM: Research last week's AI news, summarize key developments, then create a weekly briefing deck using tech-forward design.
```

These combined workflows create presentations that are both visually stunning and backed by real data.

## Tips for Effective Use

<Tip>
  **Be Specific About Style**\
  ✅ Good: "Minimalist design with earth tones and sans-serif typography"

  ❌ Vague: "Make it look good"
</Tip>

<Tip>
  **Include Context**

  ✅ Good: "Pitch deck for sustainable fashion startup targeting Gen Z consumers"

  ❌ Vague: "Fashion presentation"
</Tip>

<Tip>
  **Describe Visual Elements**

  ✅ Good: "Use photography of real products, natural lighting, lifestyle imagery"

  ❌ Vague: "Add some pictures"
</Tip>

<AccordionGroup>
  <Accordion title="Do I need design skills to use Nano Banana Pro?">
    Click the 🍌 Nano Banana Pro button in the Slides interface before entering your prompt. You'll see it next to the Slides button in the input area.
  </Accordion>

  <Accordion title="Can Manus combine research with Nano Banana Pro slides?">
    Yes. Manus can research topics, analyze data, or scrape websites, then automatically create visually stunning presentations with that information. Just describe the full workflow in your prompt.
  </Accordion>

  <Accordion title="Can I edit slides after Manus generates them?">
    Not yet, Nano Banana Pro slides are complete images with text rendered directly into the visual. For editable presentations, use Manus's standard HTML slides instead. But Manus will support this soon. 
  </Accordion>

  <Accordion title="When should I use Nano Banana Pro vs. standard HTML slides in Manus?">
    Use Nano Banana Pro for high-impact visual presentations (pitches, launches, keynotes). Use HTML slides for data-heavy content you'll update frequently or need to edit.
  </Accordion>

  <Accordion title="Does using Nano Banana Pro consume more credits?">
    Image generation uses credits based on complexity and quantity. Check your Manus plan details for specific credit costs per slide or image.
  </Accordion>
</AccordionGroup>
