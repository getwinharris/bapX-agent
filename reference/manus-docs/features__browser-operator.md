# Browser Operator

> A browser extension that transforms your web browser into an autonomous agent.

## What it is

<iframe src="https://www.youtube.com/embed/kaDwyZVFDJs" title="YouTube video player" frameborder="0" className="w-full aspect-video rounded-xl" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen />

Manus Browser Operator is a browser extension that transforms your web browser into an autonomous agent. It allows Manus to securely perform tasks on your behalf directly within your local browser, using your existing logins and sessions.

### True Autonomous Delegation

Manus is a sophisticated AI Agent that doesn't just chat—it operates. Give it a multi-step goal, and it will plan, navigate, click, and execute the entire workflow across various sites without continuous human supervision.

### Powered by Your Active Tabs

Manus Browser Operator works directly within your current browser session, securely utilizing your existing logins and active tabs to complete tasks on sites you already use and trust.

### Automate Your Most Tedious Tasks

Instantly eliminate manual clicking, form filling, and cross-platform data extraction. From complex research and booking logistics to summarizing data, Manus handles the tedious processes that take hours out of your day.

## How to use it

### 1. Connect Browser

When you assign a task requiring web access, Manus will ask for permission to control your browser. Grant access to allow the agent to take over.

### 3. Autonomous Action

Manus works in a dedicated tab to complete your task. It uses your logged-in sessions to navigate websites, gather information, and perform actions as instructed.

## What you need to know

### Local vs. Cloud Browser

The Browser Operator uses your **local browser**, which is ideal for tasks requiring access to sites where you are already logged in. This avoids CAPTCHAs and security checks. For other tasks, Manus uses an isolated **Cloud Browser**.

| Feature            | Manus Browser Operator (Local)          | Cloud Browser                          |
| ------------------ | --------------------------------------- | -------------------------------------- |
| **Environment**    | Your local desktop browser              | Isolated, sandboxed cloud environment  |
| **Authentication** | Uses your existing browser logins       | Requires logging in within the session |
| **Best For**       | Authenticated sessions, sensitive sites | General web tasks, broad research      |

### Security and Control

You have full control over the Browser Operator. You must authorize every session, and you can stop the agent at any time by simply closing the browser tab. All actions are logged for complete transparency.

## Tips

* **Use for Authenticated Sites:** Best for tasks on sites like LinkedIn, CRMs, or premium tools you are already logged into.
* **Be Specific:** Give clear, step-by-step instructions for the best results.
* **Monitor Complex Tasks:** Keep an eye on the dedicated browser tab for multi-step workflows to ensure they run smoothly.
