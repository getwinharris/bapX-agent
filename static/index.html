Features

Everything you need in one place

Not just a chat interface — a full autonomous agent workspace.

Multi-Model Chat

Seamlessly switch between any AI provider. One interface for GPT-4o, Claude, Gemini, Grok, DeepSeek, and more.

100+ Built-in Skills

From web research and coding to design and data science. Skills are user-created in your sandbox and ready to toggle on/off.
