Quick Start

bapX is a browser-based AI agent platform. Here's how to get started:

  1. Create an account — Sign up at bapX.in/signup with your email or Google account.
  2. Connect a provider — Add your OpenAI API key, connect ChatGPT via OAuth, or link your Claude account. You can connect multiple providers.
  3. Start chatting — Your agent is ready. Ask it to code, research, browse the web, or create content.
  4. Explore skills — Enable skills from the library to expand your agent's capabilities.
💡 No credit card needed. You pay nothing to bapX until you need storage. AI compute is charged through your own provider accounts.

Connecting Providers

bapX supports 26+ AI providers. You can connect them in two ways:

API Key

Go to Settings → Providers and add your API key for any supported provider: OpenAI, Anthropic, Google Gemini, DeepSeek, OpenRouter, xAI Grok, Groq, Mistral, Together AI, Cohere, Perplexity, Hugging Face, and more.

OAuth

Supported OAuth providers: ChatGPT Plus/Pro, Claude Max, Google, Nous Portal, Qwen Portal, and GitHub Copilot. Click the OAuth button and follow the device-code flow in your browser.

OAuth Setup

OAuth lets you connect your existing subscriptions without sharing your API key:

  1. Go to Settings → Providers → OAuth
  2. Click the provider you want to connect (ChatGPT, Claude, etc.)
  3. A device code will appear — visit the provided URL and enter the code
  4. Authorize the connection on the provider's site
  5. Your provider is now linked. Select it in the model dropdown to start using it.
🔒 Secure. OAuth tokens are encrypted at rest and never shared.

Sandbox & Isolation

Each user gets their own isolated Docker sandbox. Every agent action — code execution, file operations, browser automation — runs inside this sandbox.

Security guarantees:

  • Full filesystem isolation via Docker containers
  • Network configured per user — no cross-user traffic
  • Sandbox auto-cleans on session end
  • No shared state between users
  • Storage allocated per-user, billed by usage

Skills

bapX ships with 100+ built-in skills covering every domain:

  • Development: Code review, TDD, debugging, planning
  • Creative: Design, ASCII art, video, illustration
  • Data Science: Jupyter, evaluations, DSPy, W&B
  • DevOps: GitHub workflows, deployments, monitoring
  • Research: ArXiv, knowledge graphs, blog monitoring
  • Web: Browser automation, scraping, testing
  • And more: Gaming, social media, smart home, email

You can also create custom skills in your sandbox. Any workflow you run can be saved as a reusable skill.

Memory

bapX agents have persistent memory across sessions. Your agent remembers:

  • Your preferences and coding style
  • Project conventions you've set
  • Facts about your environment
  • Workflows and approaches you prefer

You can view, edit, or clear your agent's memory from Settings → Memory.

API Keys

Need to use bapX programmatically? Generate an API key from Settings → API Keys. Use it to call the proxy endpoint directly:

curl https://agent.bapx.in/v1/chat/completions \
  -H "Authorization: Bearer YOUR_BAPX_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"openai/gpt-4o","messages":[{"role":"user","content":"Hello"}]}'

Chat Completions Proxy

The proxy endpoint routes your request to any connected provider:

POST /v1/chat/completions
Authorization: Bearer <your-bapx-key>
Content-Type: application/json

{
  "model": "openai/gpt-4o",
  "messages": [...],
  "stream": true
}

Model format: provider/model_name (e.g., anthropic/claude-sonnet-4, google/gemini-2.0-flash)
